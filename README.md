# **🧠 思源知识库 AI 助手 (SiYuan Knowledge AI)**

你的第二大脑，基于私有笔记的智能问答专家。支持 PDF/Word/Excel 附件解析、自动增量同步与混合联网搜索。

## **✨ 核心功能**

* **⚡ 智能问答 (RAG)**：基于您的私有笔记内容回答问题，不再是通用的 AI，而是懂您的私人助手。  
* **📚 全能附件解析**：不仅能读 Markdown，还能深度解析笔记中的 **PDF、Word (.docx)、Excel (.xlsx)** 附件内容。  
* **🚀 自动增量同步**：智能识别笔记变更，后台静默同步，无需每次手动点击，极速省流。  
* **🌏 混合搜索增强**：支持 **“纯本地模式”**（严谨不编造）和 **“联网增强模式”**（利用通用知识补充）自由切换。  
* **📝 一键保存笔记**：AI 的精彩回答，点击气泡右下角即可一键保存为思源文档，知识不流失。  
* **📱 全平台支持**：完美适配思源笔记 **桌面端、网页端、手机 APP**，随时随地访问您的知识库。

## **🚀 快速开始**

### **1\. 安装插件**

在思源笔记的 **“集市 \-\> 插件”** 中搜索 SiYuan Knowledge AI 或 思源知识库AI助手 并安装。

### **2\. 激活服务**

1. 点击顶部工具栏右侧的 **🧠 大脑图标**。  
2. 在弹出的界面中输入您购买的 **激活码 (VIP\_xxxx)**。  
3. 点击 **“立即激活”**。

### **3\. 同步数据**

1. 激活成功后，进入设置页。  
2. **绑定笔记本**：选择您希望 AI 读取的笔记本（建议选择核心知识库）。  
3. 点击 **“⚡ 立即全量同步”**。  
   * *提示：首次同步可能需要几分钟（取决于笔记数量），后续将自动进行增量更新，无需干预。*

### **4\. 开始提问**

* 回到聊天界面，直接输入问题即可。  
* 勾选 **“🌏 联网”** 可开启混合搜索模式。  
* 点击 **“⚙️ 设置”** 可切换 AI 模型（如 Kimi、DeepSeek 等）。

## **❓ 常见问题 (FAQ)**

**Q: 我的数据安全吗？**

A: **非常安全。**

1. **数据隔离**：您的笔记在云端向量库中通过设备指纹进行严格隔离，其他用户无法检索到您的数据。  
2. **隐私保护**：我们仅存储向量化后的数据索引，不会将您的全量笔记投喂给公共大模型进行训练。  
3. **按需调用**：仅在您提问时，系统会检索最相关的 10-20 个片段发送给 AI 生成答案。

**Q: 手机端/网页端提示 "连接失败"？**

A: 本插件已适配 HTTPS 标准协议。

* 请检查手机网络是否正常。  
* 如果您是自部署的思源网页版，请确保您的浏览器信任了 SSL 证书。  
* 点击激活页面的 **“🛠️ 网络诊断”** 按钮查看具体错误。

**Q: 为什么搜不到刚写的笔记？**

A: 插件默认每 **1 分钟** 自动检查一次更新。如果您刚写完急需搜索，可以在设置里点击 **“⚡ 立即同步”**。
